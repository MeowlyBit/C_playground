# C Playground

Personal repository where I experiment with C while learning how computers work at a low level.

This project documents my journey from basic C concepts toward understanding computer architecture, embedded systems and low-level programming.

Most explanations and notes are written directly inside the source code as comments.

---

## Topics covered so far

### Module 1
- basic program structure
- printing values
- integer variables
- decimal, octal and hexadecimal literals

### Module 2
- floating point numbers
- arithmetic operators
- operator precedence and parentheses
- `char` type and ASCII operations

---

## Repository structure


C_playground
│
├─ module_1
├─ module_2
└─ ...


Each module contains small programs and experiments exploring specific C concepts.

---

## Goal of this repository

The goal is not only to learn the C language, but to gradually understand:

- how programs become machine code
- how data is represented in memory
- how computers execute instructions
- foundations of embedded systems and low-level programming
